05 · Spatial audio
Sådan lyder nærhed

Jo tættere på,
jo højere hører du dem.

3–20 tiles: lineær fade kontor-mumlen i baggrunden
Social zone: 30 tile-radius udvidet hørbarhed i køkkenet

07 · Mødelokaler
Rumreservation

Book som I plejer.
Bare smartere.

Klik på et mødelokale, vælg et tidsrum, og bookingen lander i alles Outlook-kalendre via Microsoft Graph. Vi sender en 5-minutters notifikation til alle online — i selve VirtualOffice — så ingen kommer 7 minutter for sent.

Konflikt-fri — PostgreSQL exclusion-constraint forhindrer overlap på DB-niveau.
To-vejs sync — rumreservationer fra Outlook vises i VirtualOffice og omvendt.
Stille notifikationer — pop op i kortet, ikke et email-spam.

09 · Integration
Bygget til Microsoft 365-organisationer

Jeres Azure AD og Outlook — uden flytning, uden migration.

VirtualOffice integrerer på de to punkter, der betyder noget for jeres IT-afdeling: identitet og kalender. Resten ligger pænt isoleret i vores stack.

SSO

Microsoft Entra ID

OAuth2-flow via MSAL. Brugere logger ind med jeres eksisterende konti. Auto-provisioning af nye medarbejdere via tenant-binding.

  • Delegated permissions, ingen lagrede passwords
  • Multi-tenant — kun jeres tenant kan logge ind
  • Conditional Access og MFA respekteres
CAL

Outlook & Graph API

Mødelokaler synkroniseres med jeres Exchange resource mailboxes. Bookinger findes i begge kalendre. Afbud det ene sted afbestiller det andet.

  • Place.Read.All + Calendars.ReadWrite
  • Webhook-baseret to-vejs sync
  • Krypterede refresh tokens i Azure Key Vault
